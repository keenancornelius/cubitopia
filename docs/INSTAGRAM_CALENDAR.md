# Cubitopia Instagram Launch Calendar
## 2-Week Content Strategy (Pre-Launch + Launch Week)

**Game:** Cubitopia - Turn-based voxel strategy game with 9 tribes, hex-grid combat, destructible terrain, and base building.

**Audience:** Indie game enthusiasts, strategy game fans, voxel/pixel art lovers, browser game players

**Goals:** Build hype, showcase tribe diversity, drive Discord/community signups, encourage launch day play
